## **:wrench: Developer usage**

### **Installation**

In order to install the project and all dependencies, enter in the project folder and run `install`

#### Configuration

Update your configuration file
```
PROPERTY1=value1
PROPERTY2=value2
PROPERTY3=value3
```

Then run the config script:
```sh
config project
```

start
